// Listen for messages from the popup
chrome.runtime.onMessage.addListener((request, sender, sendResponse) => {
  if (request.action === "exportChatToPDF") {
    exportChatToPDF();
    sendResponse({status: "Exporting chat to PDF..."});
  }
});

function exportChatToPDF() {
  // Load the jsPDF and html2pdf libraries dynamically
  const script = document.createElement('script');
  script.src = chrome.runtime.getURL('lib/html2pdf.bundle.min.js');
  script.onload = function() {
    // Get the chat container - adjust selector based on DeepSeek's actual HTML structure
    const chatContainer = document.querySelector('.chat-container') || 
                         document.querySelector('.conversation') || 
                         document.body;
    
    // Options for PDF export
    const opt = {
      margin: 10,
      filename: `deepseek-chat-${new Date().toISOString().slice(0,10)}.pdf`,
      image: { type: 'jpeg', quality: 0.98 },
      html2canvas: { scale: 2 },
      jsPDF: { unit: 'mm', format: 'a4', orientation: 'portrait' }
    };
    
    // Export to PDF
    html2pdf().from(chatContainer).set(opt).save();
  };
  document.head.appendChild(script);
}

Converting Web Pages to PDF with JavaScript (Styling Preserved)

There are several robust JavaScript libraries that can help you convert HTML to PDF while preserving styles. Here are the best approaches:

1. Using html2pdf.js (Recommended)

This is one of the most popular solutions that combines html2canvas and jsPDF.

Implementation:

<script src="https://cdnjs.cloudflare.com/ajax/libs/html2pdf.js/0.10.1/html2pdf.bundle.min.js"></script>

<script>
function generatePDF() {
  // Get the element you want to convert
  const element = document.getElementById('content-to-export');
  
  // Options for the PDF
  const opt = {
    margin:       10,
    filename:     'document.pdf',
    image:       { type: 'jpeg', quality: 0.98 },
    html2canvas: { 
      scale: 2,          // Higher quality
      logging: true,      // Debug logging
      useCORS: true,      // Enable cross-origin images
      allowTaint: true,   // Allow tainted images
      letterRendering: true
    },
    jsPDF:       { unit: 'mm', format: 'a4', orientation: 'portrait' }
  };

  // Generate the PDF
  html2pdf().set(opt).from(element).save();
}
</script>

Advantages:

  • Preserves most CSS styles

  • Handles complex layouts well

  • Supports images and fonts

  • Easy to use

2. Using jsPDF with html2canvas

For more control, you can use these libraries separately:

<script src="https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js"></script>

<script>
async function generatePDF() {
  const { jsPDF } = window.jspdf;
  const element = document.getElementById('content-to-export');
  
  // Create canvas from HTML
  const canvas = await html2canvas(element, {
    scale: 2,
    logging: true,
    useCORS: true,
    allowTaint: true,
    letterRendering: true
  });
  
  // Convert canvas to image
  const imgData = canvas.toDataURL('image/png', 1.0);
  
  // Calculate PDF dimensions
  const pdfWidth = canvas.width * 0.264583; // Convert px to mm
  const pdfHeight = canvas.height * 0.264583;
  
  // Create PDF
  const pdf = new jsPDF({
    orientation: pdfWidth > pdfHeight ? 'landscape' : 'portrait',
    unit: 'mm',
    format: [pdfWidth, pdfHeight]
  });
  
  pdf.addImage(imgData, 'PNG', 0, 0, pdfWidth, pdfHeight);
  pdf.save('document.pdf');
}
</script>

const puppeteer = require('puppeteer');

async function generatePDF(url, outputPath) {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  
  await page.goto(url, { waitUntil: 'networkidle0' });

      right: '20mm',
      bottom: '20mm',
      left: '20mm'

}

Troubleshooting Common Issues

  1. Missing styles:

    • Wait for all resources to load before generating PDF

    • Use window.onload or similar events

  2. Blurry text:

    • Increase the scale factor (2x or 3x)

    • Set letterRendering: true in html2canvas

  3. Cross-origin issues:

    • Enable useCORS and allowTaint

    • Consider proxying external resources

  4. Large PDFs:

    • Split content into multiple pages

    • Use page-break-inside: avoid in CSS

Understanding Global Pandemics: A Comprehensive Explanation

A global pandemic is an outbreak of an infectious disease that spreads across multiple countries or continents, affecting a large number of people. Unlike epidemics (which are confined to a specific region), pandemics have worldwide reach, often causing significant health, economic, and social disruptions.

1. What Defines a Pandemic?

A disease outbreak becomes a pandemic when:

  • It is new or re-emerging (e.g., a novel virus like COVID-19).

  • It spreads easily (high human-to-human transmission).

  • It affects multiple countries or continents.

  • It causes widespread illness or death.

Examples of Historical Pandemics:

  • Black Death (1347–1351) – Bubonic plague killed ~75–200 million people.

  • Spanish Flu (1918–1919) – H1N1 influenza killed ~50 million.

  • HIV/AIDS (1981–present) – Ongoing, with ~40 million deaths.

  • COVID-19 (2019–present) – Caused by SARS-CoV-2, over 7 million deaths.

2. How Pandemics Spread

Pandemics typically follow these stages:

  1. Animal-to-Human Transmission (Spillover Event)

    • Many pandemics originate in animals (zoonotic diseases).

    • Examples: COVID-19 (possibly bats), HIV (chimpanzees), H1N1 (pigs).

  2. Localized Outbreak

    • The disease spreads in a small region (e.g., Wuhan for COVID-19).

  3. Community Spread

    • Human-to-human transmission accelerates.

  4. Epidemic (Regional Spread)

    • The disease spreads across cities or countries.

  5. Pandemic (Global Spread)

    • The WHO declares a pandemic when multiple continents are affected.

Transmission Routes:

  • Respiratory droplets (COVID-19, flu)

  • Direct contact (HIV, Ebola)

  • Contaminated surfaces (Norovirus)

  • Vector-borne (Mosquitoes for Zika, Malaria)

3. Factors That Accelerate Pandemics

A. Globalization & Travel

  • Air travel allows viruses to spread globally in days (COVID-19 reached 100+ countries in 3 months).

  • Urbanization increases close human contact.

B. Weak Healthcare Systems

  • Countries with poor medical infrastructure struggle to contain outbreaks.

C. Vaccine Hesitancy & Misinformation

  • Anti-vaccine movements slow herd immunity (e.g., measles resurgence).

D. Climate Change

  • Rising temperatures expand habitats for disease-carrying mosquitoes (e.g., Dengue, Zika).

E. Animal-Human Encroachment

  • Deforestation and wildlife trade increase spillover risks (e.g., Ebola from bushmeat).

4. The Impact of Pandemics

A. Health Consequences

  • High death tolls (COVID-19 killed more than WWI in some countries).

  • Overwhelmed hospitals (shortages of beds, ventilators, doctors).

  • Long-term health effects (Long COVID, post-viral syndromes).

B. Economic Disruptions

  • Stock market crashes (2020 saw the fastest drop in history).

  • Supply chain breakdowns (shortages of food, electronics, medicine).

  • Mass unemployment (lockdowns closed businesses).

C. Social & Psychological Effects

  • Lockdowns & isolation led to mental health crises (depression, anxiety).

  • Stigma & discrimination (e.g., anti-Asian hate during COVID-19).

  • Education disruptions (school closures affected 1.6 billion students in 2020).

5. How Governments & Organizations Respond

A. Containment Measures

  • Quarantines & lockdowns (China’s strict COVID lockdowns).

  • Travel bans (U.S. restricted flights from Europe in 2020).

  • Contact tracing (South Korea used phone data to track infections).

B. Medical Responses

  • Vaccine development (COVID-19 vaccines were developed in record time).

  • Antiviral treatments (Paxlovid for COVID-19, Tamiflu for flu).

  • Public health campaigns (mask mandates, handwashing).

C. Global Coordination

  • World Health Organization (WHO) declares pandemics and advises policies.

  • COVAX distributed COVID vaccines to poorer countries.

6. Can We Prevent Future Pandemics?

A. Early Detection Systems

  • AI-powered disease surveillance (e.g., tracking outbreaks via social media).

  • Faster genetic sequencing (identifying new viruses quickly).

B. Better Preparedness

  • Stockpiling medical supplies (masks, ventilators).

  • Training healthcare workers for surge capacity.

C. Reducing Zoonotic Risks

  • Regulating wildlife markets.

  • Monitoring deforestation and animal migration.

D. Strengthening Global Cooperation

  • Sharing vaccine patents.

  • Improving WHO funding and authority.

7. The Next Pandemic: What Experts Fear

  • Disease X – WHO’s term for an unknown future pathogen.

  • Antibiotic-resistant bacteria (could make infections untreatable).

  • Climate-driven pandemics (melting permafrost may release ancient viruses).

Final Thoughts

Pandemics are inevitable, but their impact can be reduced through science, preparedness, and global cooperation. COVID-19 was a wake-up call—will we be ready for the next one?
